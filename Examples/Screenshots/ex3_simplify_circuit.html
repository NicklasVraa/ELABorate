simplify(circuit)

This function attempts to simplify the circuit, updating the entire circuit object.
This loads a circuit from a text file and displays its netlist. The circuit is then analyzed and some results are displayed.
clc; clear;
path = 'Circuits/series_parallel_2.txt';
circuit = Circuit(path);
circuit.get_netlist
ans =
'V1 1 0 AC V1 R1 1 2 1000 R2 2 0 2000 R3 2 0 2000 R4 1 2 3000 R5 1 0 1000 C1 1 0 2 C2 0 1 3 '
ELAB.analyze(circuit)
Symbolic analysis successful (0.690777 sec).
circuit.symbolic_node_voltages
ans =
