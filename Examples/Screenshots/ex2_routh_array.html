routh(coeffs, eps)

The function takes an array of coefficients representing a characteristic equation and the number eps, which may replace zeros.
For example, obtain the characteristic equation of a system, then compute its Routh-array for stability analysis.
clear; syms s K_p K_i K;
sys1 = K/(s*(s+1)*(s+2)) + 1; % Unity gain system.
sys2 = 1 + K_p*(2/(s^3 + 4*s^2 + 5*s + 2)); % P-controlled system.
K_p = 3; % Keeping one parameter constant.
sys3 = 1 + (K_p + K_i/s)*(2/(s^3 + 4*s^2 + 5*s + 2)); % PI-controlled system.
coeffs = ELAB.sd2ce(sys3)
